# 🚀 Space Survivor

Space Survivor é um jogo 2D de sobrevivência no espaço desenvolvido com Unity.
O objetivo é simples: desviar de obstáculos e sobreviver o máximo de tempo possível.

## 🎮 Sobre o jogo

Você controla uma nave em um ambiente espacial onde meteoros surgem constantemente.
A dificuldade aumenta com o tempo, exigindo reflexos rápidos e precisão.

## 🌐 Versão Web

Este projeto está disponível na versão Web (Unity WebGL).

Para facilitar a execução local, a compressão foi desativada, permitindo que o jogo rode em servidores simples.
